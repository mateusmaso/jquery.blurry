jquery.blurry
=============

This library is an extension for jQuery which allows blurring images using canvas technology.

## Features

* Convert image into canvas.
* Smooth ```blurry``` function with strength control. 

## Dependencies

* jquery.js (>= 2.0)

## Examples

```javascript
$('image').blurry(5);
```
